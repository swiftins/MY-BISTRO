Для безопасного хранения и использования API-ключа Telegram-бота вне исходного кода, можно воспользоваться несколькими методами. Вот пошаговое руководство по каждому из них:

### 1. Использование файла `.env` и библиотеки `python-dotenv`

1. **Установите библиотеку `python-dotenv`:**

   Убедитесь, что у вас установлена библиотека `python-dotenv`. Если нет, установите её с помощью pip:

   ```bash
   pip install python-dotenv
   ```

2. **Создайте файл `.env`:**

   В корне вашего проекта создайте файл с именем `.env`. В этом файле сохраните API-ключ в следующем формате:

   ```
   TELEGRAM_API_KEY=ваш_телеграм_api_ключ
   ```

3. **Загрузите API-ключ в вашем Python-коде:**

   В коде вашего бота загрузите переменные окружения из файла `.env` с помощью `load_dotenv()`:

   ```python
   from dotenv import load_dotenv
   import os

   # Загрузить переменные окружения из файла .env
   load_dotenv()

   # Получить API-ключ
   telegram_api_key = os.getenv('TELEGRAM_API_KEY')

   # Используйте API-ключ для инициализации бота
   # bot = telegram.Bot(token=telegram_api_key)
